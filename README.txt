Этот тестовый проект создан на чистых PHP и Javascript, никакие сторонние библиотеки не были использованы.
Для запуска проекта после копирования скриптов в Apache необходимо настроить конфигурацию:
  в файле /app/config/config.ini изменить параметры подключения к БД;
  установить БД, запустив в соответствующей утилите скрипт /app/backup/db.sql;
  для корректной работы сайта желательно в файле /app/config/config.ini изменить имя домена на то, 
куда вы поместили содержимое архива.
